# サムネイル画像生成プロンプト集(全22技術)

## 共通設定(全プロンプト先頭に付ける)

```
Photorealistic cinematic editorial photograph for a technology magazine, 16:9 wide shot,
dramatic lighting, high detail, no text, no logos, no watermarks, no UI overlays
```

- 推奨アスペクト比: **16:9**(カードは横長トリミングのためこれが最適)
- サイトのモノトーン基調に寄せたい場合は末尾に `muted color palette, deep blacks, low saturation` を追加
- 生成後は `assets/thumbs/<ID>.jpg` の名前で保存してください(下記ID)。渡していただければ表示側は私が配線します
